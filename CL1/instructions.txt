1. Download and install Anaconda Python 3.6 https://www.continuum.io/downloads
2. Install Git, https://git-scm.com/book/en/v2/Getting-Started-Installing-Git.
3. Clone the github repo for the course inputting the following command in a terminal (this creates a folder in your current directory with the necessary files for the next instructions).    
      git clone https://github.com/JulianoLagana/deep-machine-learning.git
4. Navigate to the folder created in 3 (use the cd command in your terminal).
5. Create the Anaconda environment for the course by typing (in the terminal)
      conda env create -f environment.yml
6. Wait for the installations to complete.
7a. Then, if you are using Mac or Linux, type
      source activate dml
7b. Then, if you are using Windows, type
      activate dml
8.  It is time to start using Jupyter notebook. To do so, type 
      jupyter notebook
9.  Have a look at the two Jupyter notebooks, testing_notebook.ipynb and Getting started with JN.ipynb, available in this folder. The first file checks if the installation was successful, whereas the second file give you initial advices regarding how to use Jupyter notebooks.
10. To deactivate the virtual environment, you simply type source deactivate or deactivate, depending on if you are using Mac/Linux or Windows.
